# Msia_Sign_Language_Recognition

**For (AI Developers)**
Follow the below steps to setup connection to dataset.
1. Make sure your google drive Shared With Me section has the dataset shared through the link => https://drive.google.com/drive/folders/145qAw4rhdvr0m8ZAar-B3AgyzXhQDMoE?usp=sharing

2. Add a shortcut to the dataset in your Google Drive account:
   - Open the Google Drive account.
   - Navigate to the "Shared with me" section.
   - Locate the shared dataset file or folder.
   - Right-click on it and select "Add shortcut to Drive."
   - Choose "My Drive" and click "Add Shortcut." This creates a shortcut to the shared data within your Colab-associated Google Drive, making it accessible as if it were directly in your Drive.

3. To open the .ipynb file in the GitHub, first open any .ipynb file to open the Google Colab interface. (Only for AI Developers)

4.  Then go to File > Open notebook > GitHub (on left panel), fill the following info
   - user : raddified
   - repository : raddified/Msia_Sign_Language_Recognition
   - branch : data_preprocessing / model_optimization
   
5. Select your working file.
